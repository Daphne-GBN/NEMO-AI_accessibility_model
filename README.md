# DyslexEdu

AI-Based Accessibility Tool for Dyslexic Students

## Overview
DyslexEdu is an AI-powered educational accessibility tool designed to assist dyslexic learners in understanding text-heavy study material. The system simplifies complex educational content, provides text-to-speech assistance, and offers an adaptive reading interface to improve comprehension and learning engagement.

The platform integrates Natural Language Processing (NLP), speech processing, and accessibility-focused UI design to create a personalized learning environment for students with dyslexia.

## Key Features
- AI-powered text simplification
- Text-to-Speech (TTS) for audio learning
- Dyslexia-friendly interface
- Word highlighting during reading
- Voice interaction support
- Personalized learning assistance
